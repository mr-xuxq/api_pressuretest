# 一、框架介绍

| 代码框架     | Python+pytest     | 基于Python语言，使用pytest库来管理用例（主要用来识别和分类） |
| ------------ | ----------------- | ------------------------------------------------------------ |
| **用例管理** | **Excel**         | 使用Excel来管理用例，使工作人员无需接触代码即可执行任意API测试 |
| **集成工具** | **Jenkins**       | 将测试用例和测试报告集成到Jenkins平台，提供一键执行功能      |
| **测试报告** | **Allure Report** | Python支持的测试报告库，Jenkins亦有其插件，能很好地衔接两者，生成美观简洁的测试报告 |

Python+pytest： 基于Python语言，使用pytest库来管理用例（主要用来识别和分类）

Excel： 使用Excel来管理用例，使工作人员无需接触代码即可执行任意API测试

Jenkins： 将测试用例和测试报告集成到Jenkins平台，提供一键执行功能

Allure Report： Python支持的测试报告库，Jenkins亦有其插件，能很好地衔接两者，生成美观简洁的测试报告
